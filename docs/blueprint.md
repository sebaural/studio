# **App Name**: Heritage Hub

## Core Features:

- Profile Creation: Users can create profiles for family members, including names, birth dates, photos, and short biographies.
- Family Tree Generation: Automatically generate a visual family tree based on the relationships defined in the profiles.
- Relationship Management: Define relationships between family members, such as parent-child, sibling, and spouse.
- AI-Powered Historical Insights: Use a generative AI tool to provide historical context and insights related to family names, birthplaces, and significant life events of family members.

## Style Guidelines:

- Primary color: Muted olive green (#A3B18A), evoking a sense of history and connection to the past.
- Background color: Light beige (#F5F5DC), providing a neutral and classic backdrop.
- Accent color: Warm gold (#D4AF37), used for highlights and interactive elements to suggest legacy.
- Body text: 'Literata', a transitional serif with a literary feel.
- Headline text: 'Literata', a transitional serif with a literary feel.
- Use classic, simple icons for navigation and profile elements.
- Clean and intuitive layout with a focus on visual clarity for easy navigation and readability.
